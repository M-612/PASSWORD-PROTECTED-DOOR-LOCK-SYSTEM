## Description
This project implements a password-protected door lock system using:
- Arduino
- 4x3 Keypad
- 16x2 LCD
- Servo Motor
- Buzzer and LEDs

The user enters a password using the keypad. If correct, the servo unlocks the door.
Multiple wrong attempts trigger warning messages and buzzer alerts.

## Features
- Masked password input
- Servo-based locking mechanism
- Buzzer alert for wrong attempts
- LCD feedback messages
- Security delay after multiple failures

## Components Used
- Arduino Uno
- 4x3 Matrix Keypad
- 16x2 LCD
- Servo Motor
- Buzzer
- LEDs
